# 🧠 Lab 3 – Hệ Thống Nhúng

## 🎯 Giới thiệu
Mục tiêu của Lab 3 là làm quen với hệ điều hành thời gian thực (RTOS) và cách
sử dụng RTOS để điều khiển các tác vụ trên ESP32. Lab sẽ hướng dẫn cài đặt RTOS
trên ESP32, xây dựng các tác vụ (tasks) và quản lý chúng để thực hiện các chức năng
khác nhau, như điều khiển LED hoặc đọc cảm biến. Ngoài ra, bạn cũng sẽ học cách
chạy một mô hình học máy (Tiny ML model) trên ESP32 để thực hiện các dự đoán
trong thời gian thực, chẳng hạn như phân loại tín hiệu hoặc nhận diện đối tượng.
